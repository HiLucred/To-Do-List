# ToDo List em React

Aplicação desenvolvida no módulo "Ignite" proposto pela RocketSeat.

🛠️ Skils: TypeScript, React, HTML, CSS


## 📋 Clone o repositório
$ git clone https://github.com/HiLucred/ToDo-List-REACT


## 🔧 Instale as dependências
$ npm install

 
## ⚙️ Rode o projeto
$ npm run dev
